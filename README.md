# LoyaltyHub — Freelance Project

This repository contains the LoyaltyHub project, a full-stack loyalty management system built with Next.js, NestJS, and PostgreSQL.

## 🚀 Project Structure

- [**loyalty-system/**](./loyalty-system/) - Main application directory
  - [**frontend/**](./loyalty-system/frontend/) - Next.js 14 Progressive Web App (PWA)
  - [**backend/**](./loyalty-system/backend/) - NestJS REST API
  - [**docs/**](./loyalty-system/docs/) - Comprehensive project documentation

## 🛠️ Quick Start

For detailed setup and installation instructions, please refer to the [loyalty-system README](./loyalty-system/README.md).

### Prerequisites
- Node.js 18+
- PostgreSQL database
- npm

### Installation Overview
1.  **Backend**: Configure `.env`, run `npm install`, and start the NestJS server.
2.  **Frontend**: Configure `.env.local`, run `npm install`, and start the Next.js dev server.
